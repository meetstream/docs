---
title: 'Create Bot'
description: 'Creates a new bot instance to join a meeting'
---

<Note>
  This endpoint requires API Key authentication via the Authorization header.
</Note>

## Request

<CodeGroup>
<CodeGroupItem title="cURL">

```bash
curl -X POST "${process.env.NEXT_PUBLIC_API_URL}/bots/create_bot" \
  -H "Authorization: Token YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "meeting_link": "",
    "audio_required": true,
    "video_required": false,
    "bot_message": "",
    "live_audio_required": {
        // "websocket_url": ""
    },
    "live_transcription_required": {
        // "webhook_url": ""
    },
    "transcription": {
        "deepgram": {
            "model": "",
            "language": ""
        }
    },
    "custom_attributes": {
        "tag": "",
        "sample": ""
    },
    "callback_url": ""
}'
```

</CodeGroupItem>
</CodeGroup>

## Response

<ResponseField field="status" type="number">
  201
</ResponseField>

<ResponseField field="body" type="object">
  ```json
  {
    "bot_id": "string"
  }
  ```
</ResponseField>
