---
title: 'Remove Bot'
description: 'Removes a bot from its meeting and deletes its associated data'
---

<Note>
  This endpoint requires API Key authentication via the Authorization header.
</Note>

## Path Parameters

<ParamField field="bot_id" type="string">
  The unique identifier of the bot
</ParamField>

## Request

<CodeGroup>
<CodeGroupItem title="cURL">

```bash
curl -X GET "${process.env.NEXT_PUBLIC_API_URL}/bots/{bot_id}/remove_bot" \
  -H "Authorization: Token YOUR_API_KEY"
```

</CodeGroupItem>
</CodeGroup>

## Response

<ResponseField field="status" type="number">
  200
</ResponseField>

<ResponseField field="body" type="object">
  ```json
  {
    "success": true,
    "message": "Bot bot_123 has been removed successfully."
  }
  ```
</ResponseField>

## Error Responses

<ResponseField field="status" type="number">
  401
</ResponseField>

<ResponseField field="body" type="object">
  Unauthorized (Invalid API Key)
</ResponseField>

<ResponseField field="status" type="number">
  404
</ResponseField>

<ResponseField field="body" type="object">
  Bot not found
</ResponseField>

<ResponseField field="status" type="number">
  500
</ResponseField>

<ResponseField field="body" type="object">
  Internal Server Error
</ResponseField>
